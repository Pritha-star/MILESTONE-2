#  Resume Analyzer 

# Introduction

The Resume Analyzer is a Flask-based web application designed to evaluate and analyze resumes uploaded by users. The system examines resume content, extracts text from PDF files, identifies relevant skills, evaluates resume quality, and provides structured feedback including strengths, weaknesses, and suggestions for improvement.

This project is built as an academic and practical implementation of backend development using Python Flask, along with frontend technologies like HTML, CSS, and JavaScript.

The primary goal of this project is to help students and job seekers improve their resumes and make them more ATS (Applicant Tracking System) friendly.

---

# Objectives

- To build a functional web application using Flask
- To implement PDF parsing and text extraction
- To analyze resume content programmatically
- To identify skills and missing skills
- To provide structured improvement suggestions
- To understand Git & GitHub workflow
- To implement login and file upload system
